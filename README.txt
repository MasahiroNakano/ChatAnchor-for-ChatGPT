ChatGPT Navigator + Scroll Lock

Files:
- manifest.json
- page-bridge.js
- content.js

Install:
1. Open chrome://extensions
2. Enable Developer mode
3. Click Load unpacked
4. Select this folder

Shortcuts:
- Alt + Up: previous user message
- Alt + Down: next user message
- Alt + L: toggle Stay/Follow

Notes:
- This extension only changes the page UI locally on chatgpt.com.
- It does not send your chat data anywhere.
- If ChatGPT changes its DOM structure, the message selector logic may need an update.
